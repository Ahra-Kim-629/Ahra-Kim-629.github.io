🧩 JavaScript, HTML, and CSS - What’s the difference?

Think of a website like a house. HTML is the structure (walls, windows), CSS is the style (paint, furniture), and JavaScript is what makes things move (doors that open, lights that turn on).

🔁Control Flow & Loops

Control flow is the order in which instructions run. Loops help us repeat actions. Imagine packing lunchboxes: for each child, you add a sandwich, a drink, and fruit. That’s a loop! 


    for (let i = 0; i < 3; i++) {
    console.log("Packed lunchbox #" + (i + 1));
    }

🌳What is the DOM?

The DOM is like a map of your webpage that JavaScript can read and change. You can use it to change text, colors, or respond to clicks! 


    document.querySelector("h1").textContent = "Hello from JavaScript!";

📦Arrays vs Objects

Arrays are like shopping lists (ordered), and objects are like labeled boxes. 


    let fruits = ['apple', 'banana'];
    fruits[0]; // 'apple'

    let person = { name: 'Eunice', age: 28 };
    person.name; // 'Eunice'

🛠️What are Functions and why use them?

Functions are reusable chunks of code, like recipes. You write it once and use it whenever needed. 


    function greet(name) {
    console.log("Hello, " + name);
    }

    greet("Soojin"); // Hello, Soojin
