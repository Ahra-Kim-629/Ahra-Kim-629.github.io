What was the problem?



I was writing a capitalize function that should return a string with the first letter of each word in uppercase. I used .map() and .toUpperCase() to convert all the letters, and joined them back together using .join(''), but I forgot to add a space ' ' between the words. So my test kept failing because the words were all stuck together.



What problem-solving techniques did you use?



I read the error message carefully and compared the expected result to my output. I used console.log() to check what my function was returning. Then I read the test file more closely and realized that the expected output had spaces between words. After that, I checked my .join() method and changed .join('') to .join(' ').



How did you feel throughout the process?



At first, I felt frustrated because it seemed like such a small issue, but I couldn‚Äôt figure out what was wrong. Once I found the missing space, I felt relieved and proud that I solved it by carefully checking each step.



What did you learn?



I learned that even small details, like spaces in strings, can make a big difference. I also learned the importance of reading test messages and checking the expected output closely.

I first tried using a for loop, but then I remembered the .filter() method in JavaScript. I used array.filter() to check if each number was even by using the condition num % 2 === 0. It worked perfectly and made the code much shorter and cleaner.



How did you feel throughout the process?



I felt confident and excited. It was a nice moment because I realized I was starting to think like a developer‚Äîchoosing the right tool for the job.



What did you learn?
